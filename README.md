# ergo-website

Need latest version nodejs and yarn.

Need Hugo [0.55.4](https://github.com/gohugoio/hugo/releases) or later

Need Yarn setup 

# Yarn setup

```sh
$ yarn install
```

# Up server on localhost:1313

```sh
$ hugo server -D
```

# Watch assets js

```sh
$ yarn run watch
```

# Build site 

```sh
$ make site
```
